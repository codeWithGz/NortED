# NortED
 Dando um norte acadêmico para quem está perdido na rotina de estudos, utilizando um comitê de algoritmos de aprendizado de máquina.
 
## 🤖 Modelagem e Comitê de IAs
O coração do projeto consiste em um sistema de *Ensemble Learning* que utiliza **Voto Majoritário (Hard Voting)** combinando três modelos:
* Árvore de Decisão
* K-Vizinhos Mais Próximos (KNN)
* Naive Bayes
